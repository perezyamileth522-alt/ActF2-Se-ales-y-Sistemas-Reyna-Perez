DESCRIPCIÓN DEL PROYECTO

Este proyecto tiene como objetivo analizar señales en el dominio del tiempo y en el dominio de la frecuencia mediante la Transformada de Fourier.
Se realiza una simulación en Python para visualizar cómo se descomponen diferentes señales en sus componentes frecuenciales.

Incluye además materiales complementarios como un documento en Word y una presentación en PowerPoint para explicar los resultados obtenidos.

OBJETIVOS
*Generar señales en el dominio del tiempo.
*Aplicar la Transformada de Fourier para obtener su representación en frecuencia.
*Visualizar y comparar ambas representaciones.
*Interpretar el comportamiento de las señales.

CONCEPTOS UTILIZADOS

-Señales senoidales
-Ruido aleatorio
-Transformada Rápida de Fourier (FFT)
-Dominio del tiempo vs dominio de la frecuencia
-Análisis espectral

TECNOLOGÍAS UTILIZADAS

*Python 3
*NumPy
*Matplotlib
*SciPy
